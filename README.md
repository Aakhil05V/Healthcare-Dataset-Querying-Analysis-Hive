# Healthcare Dataset Querying and Analysis using Hive

A comprehensive web-based platform for healthcare dataset querying and analysis using Apache Hive with a Flask backend and an interactive HTML5 frontend.

## Features

- **Web Interface**: Single HTML5 page with integrated CSS and JavaScript
- **Backend API**: Flask-based REST API for Hive query execution
- **Data Analysis**: Support for complex healthcare data queries
- **Real-time Results**: Instant query execution and result display
- **Responsive Design**: Mobile-friendly interface

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Python Flask
- **Data Processing**: Apache Hive
- **Database**: Hadoop Distributed File System (HDFS)

## Installation

1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the Flask application: python app.py
4. Open your browser: http://localhost:5000
